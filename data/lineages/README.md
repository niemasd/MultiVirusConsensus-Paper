# Data
The benchmark experiment uses complete SARS-CoV-2 genome assemblies from [NCBI Virus](https://www.ncbi.nlm.nih.gov/labs/virus/vssi/):

* [`SC2.assemblies.fas.xz`](SC2.assemblies.fas.xz) contains 11,786 SARS-CoV-2 genome sequences from many different lineages
* [`reference.fas`](reference.fas) is the SARS-CoV-2 reference genome (`NC_045512.2`)
